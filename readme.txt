== Description ==
Saksh private IELTS preparation offer you an application using that you can provide practise tests to your students. 

This include questions set so you don't need to import any database you only need to create a page and add shortcode  [PrivateIELTSEXCERCISE] 


Student can click and see the tests and read.  

To start using this you only need two step 1 activate plugin and 2nd create a page and type the shortcode [PrivateIELTSEXCERCISE] .This all plugin will do the rest.

 

###Key Features 

£ Database included
£ Around 2800 datasets.
